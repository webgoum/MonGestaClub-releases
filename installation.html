Première ouverture du logiciel

MonGestaClub n'est pas signé par les magasins d'applications. Au tout premier lancement, votre système peut afficher un avertissement de sécurité : c'est normal et sans danger. Voici comment l'autoriser, une seule fois.

macOS

« MonGestaClub est endommagé » — la vraie solution

Ce message trompeur apparaît parce que l'application n'est pas notariée par Apple. L'app n'est pas endommagée : il suffit de retirer la mise en quarantaine ajoutée au téléchargement.

  1. Ouvrez le fichier .dmg téléchargé, puis glissez MonGestaClub dans le dossier Applications.
  2. Ouvrez l'application Terminal (Spotlight : ⌘ + Espace, tapez « Terminal », Entrée).
  3. Copiez-collez cette commande dans le Terminal, puis appuyez sur Entrée :
    sudo xattr -cr /Applications/MonGestaClub.app

    Le Terminal demandera le mot de passe de votre Mac : tapez-le puis appuyez sur Entrée (par sécurité, rien ne s'affiche pendant la saisie, c'est normal).

  4. Lancez MonGestaClub depuis le dossier Applications. C'est fait — l'avertissement ne reviendra plus.

Astuce : si vous préférez ne pas utiliser le Terminal, faites un clic droit sur l'app → Ouvrir. Si le message « endommagé » persiste malgré tout, la commande ci-dessus reste la méthode fiable.

Windows

« Windows a protégé votre ordinateur »

SmartScreen affiche cet avertissement pour les applications peu téléchargées. L'installeur est sûr.

  1. Lancez le fichier MonGestaClub Setup .exe téléchargé.
  2. Sur l'écran bleu SmartScreen, cliquez sur « Informations complémentaires ».
  3. Cliquez sur « Exécuter quand même », puis suivez l'installation.
Linux

Rendre l'AppImage exécutable

Le fichier .AppImage doit être rendu exécutable une seule fois avant son premier lancement.

  1. Ouvrez un terminal dans le dossier où se trouve le fichier MonGestaClub-1.0.6-linux.AppImage téléchargé.
  2. Copiez-collez cette commande, puis appuyez sur Entrée :
    chmod +x MonGestaClub-1.0.6-linux.AppImage
  3. Double-cliquez sur le fichier (ou lancez-le depuis le terminal avec ./MonGestaClub-1.0.6-linux.AppImage). C'est fait — l'application démarre normalement.
